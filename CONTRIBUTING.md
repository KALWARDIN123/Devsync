# Contributing

Thank you for your interest in contributing to Devsync.

Steps to contribute:

- Fork the repository and create a feature branch from `main`.
- Run the test suite locally: `python -m pip install -r requirements.txt && python manage.py test`.
- Make code changes and ensure tests pass.
- Format code with your preferred tools and keep changes focused.
- Open a pull request describing the change and link any related issues.

By contributing, you agree to follow the project's Code of Conduct.
